# advancedvison
千葉工業大学大学院未来ロボティクス専攻 2025年度 アドバンスドビジョン課題用リポジトリ．

## 概要
本モデルは28×28のグレースケール画像を入力とし，中間層でわずか4次元まで情報を圧縮した後，元の画像を復元するオートエンコーダモデルです．

本モデルの性能評価としてMNISTデータセットを使用して10-epoch学習をした結果，784次元の情報を約1/200の4次元に圧縮しても，数字の概形を復元可能であることを確認しました．
